# 🏥 Libby's Habbadabba Sheet

**Activity Attendance Tracker** — an Excel workbook for tracking patient activity attendance across multiple units. Same functionality as the web dashboard, but as a plain `.xlsx` file that works on any work computer with no IT approval needed.

## Features

- **3 unit sheets** — Unit 1 (North), Unit 2 (South), Unit 3 (East)
- **Dropdown selection** — Weekdays: 0, 45, 90 mins. Weekends: 0, 45, 90, 120, 240, 330 mins
- **Auto-calculated weekly totals** — SUM formulas in each row
- **Frozen header row** — headers stay visible when scrolling
- **Print-friendly** — landscape layout, fits to page width
- **No macros, no VBA** — pure Excel data validation + formulas. IT won't flag it.

## Usage

1. Open `Activity_Attendance_Tracker.xlsx` in Excel
2. Pick a unit tab at the bottom
3. Click each day cell and choose attendance minutes from the dropdown
4. Weekly total calculates automatically
5. Print or share the file as needed

## IT Safe

- No macros, no VBA, no scripts
- No external connections
- Just formulas and dropdowns — exactly like any normal Excel file
- Won't trigger security alerts or fake-phishing tests
